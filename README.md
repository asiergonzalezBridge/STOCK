# Gestión de Stock

## Descripción

Gestión de Stock es un prototipo de aplicación web pensado para ayudar a pequeños comercios a gestionar mejor sus materias primas y prever las compras necesarias según la demanda de sus productos.

Muchos negocios que trabajan con materias primas deben estimar las ventas para calcular cuánto comprar. Esto puede provocar dos problemas habituales: comprar demasiado y que los productos se estropeen, o comprar demasiado poco y perder ventas por falta de stock.

Este proyecto propone una herramienta sencilla que permita registrar productos, estimar la demanda y calcular automáticamente la cantidad de materias primas necesarias para cubrir las ventas previstas, ayudando así a optimizar la gestión del negocio.

## Estructura del Prototipo

Explicación de las vistas creadas y cómo se navega entre ellas.

* **index.html:**  
Es la página principal de la aplicación. En ella se presenta la idea del proyecto y las ventajas de utilizar un sistema de gestión de stock. La página muestra diferentes tarjetas informativas que explican cómo la herramienta puede ayudar a los comercios a optimizar sus compras, ahorrar tiempo y mejorar el control de su inventario.

* **funcion.html:**  
Esta vista explica cómo funciona la aplicación. Se muestran ejemplos visuales y explicaciones sobre cómo el sistema puede ayudar a calcular las compras necesarias de materias primas según la demanda estimada de los productos.

* **contacto.html:**  
Esta página simula una pantalla de contacto o solicitud de demostración del producto. Permite que un usuario interesado en la herramienta pueda solicitar más información o una demo del sistema.

## Enfoque Técnico

Breve resumen de cómo se ha planteado el desarrollo:

* **Diseño Mobile First**, priorizando que la web funcione correctamente en dispositivos móviles antes de adaptarse a pantallas más grandes.
* **Uso de selectores básicos y combinados** en CSS para organizar y aplicar estilos de forma clara.
* **Uso de diferentes valores de display**, principalmente `grid`, para estructurar la maquetación de la página.
* **Uso de variables CSS** para mantener consistencia en colores, tamaños y espaciados en todo el proyecto.
* **Interactividad básica**, como efectos hover en enlaces y navegación entre páginas.
